Applikation Kontaktliste

Beschreibung

Diese Applikation weist eine Kontaktliste auf. Folgende Informationen über eine Person können gespeichert werden: Vorname, Nachname, Spitzname, E-Mail Adresse, Telefonnummer und die Wohnadresse.

Architektur

Die Applikation is modular aufgebaut. Die einzelnen Module können ausgetauscht werden. Die Applikation wird mit Hilfe Maven installiert und gestartet. Zudem lässt sich ein jar-File für jedes Modul generieren bei der Installation der Applikation.

Funktionalitäten

Here are the functionality this website offers:

  • Neue Kontakte können hinzugefügt werden
  • Neue Kontakte können hinzugefügt werden
  • Einzelne Kontakte können angezeigt werden
  • Einzelne Kontakte können editiert werden
